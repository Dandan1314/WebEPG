
## 使用指南

#### 1.给需获取焦点的元素设置相同的类名，如：class="item"

#### 2.引入js
       <script type="text/javascript" src="../js/FindFocus.js"></script>

#### 3.设置类名  
       默认类名为 "item",如果要设置的类名为 "item" 可不调用下面代码。
       FindFocus.FocusKeyEvent.setClassName(".item") //不要忘记输入"."
